
# CompuWork
Sistema de gestión de empleados y departamentos en Java, implementado con arquitectura MVC y Swing.

## 🚀 Funcionalidades
- Crear empleados permanentes y temporales
- Crear departamentos
- Listar empleados
- Generar reportes de desempeño
- Validar datos con bloques `try-catch`

## 🛠️ Tecnologías utilizadas
- Java (NetBeans)
- Swing (interfaz gráfica)
- Arquitectura MVC
- Git & GitHub


## 📂 Estructura del proyecto
- `src/models` → Clases del modelo (Empleado, Departamento, ReporteDesempeno)
- `src/views` → Interfaz gráfica con Swing
- `src/controllers` → Controlador que conecta vista y modelo
- `nbproject` → Configuración de NetBeans
- `build.xml` → Script de construcción

## ▶️ Cómo ejecutar
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/danielvillegas-code/CompuWork.git

2. Abrir el proyecto en NetBeans.

3. Ejecutar la clase VistaSwing.
